Distribuições Java

O programa Java é distribuído em duas edições: JDK e JRE.
  • JRE
    • Formada apenas pela máquina virtual Java e seus componentes operacionais.
    • É usado apenas para executar aplicações Java em um computador.
  • JDK
    • Ferramenta para desenvolvimento de programas codificados em Java.
    • É formado pelo JRE + kit de desenvolvimento.

Instalação do programa

A instalação do pacote Java SE ocorrerá de maneira diferente em cada sistema operacional.
  • Windows e Linux
    • Precisa instalar o Java SE e configurar as variáveis de ambiente.
  • Mac OS
    • Precisa apenas instalar o Java SE.

Variáveis de ambiente

  • JAVA_HOME
    • Indica ao sistema o local onde o programa Java foi instalado.
  • PATH
    • Indica o local onde se encontram os arquivos binários (bytecode) do programa.
